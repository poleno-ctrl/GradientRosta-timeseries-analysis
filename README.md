# Прогноз РТО магазинов Пятёрочки (этап А — «Больше данных»)

Прогноз розничного товарооборота (РТО) магазинов сети на **март 2025**.

## Задача
- **Метрика:** MAPE (чем меньше, тем больше баллов; см. [docs/task_description.md](docs/task_description.md)).
- **Посылка:** `test.csv` с колонками `new_id`, `rto` — **18657** строк (по числу магазинов).
- **Ограничения:** размер файла ≤ 1 МБ, до 100 попыток.

## Данные
- `data/raw/train_2.csv` — история РТО по месяцам + соц.-гео. признаки магазинов.
- Покрытие: **2023-01 … 2025-02** (~485k строк). Цель — **2025-03**.
- Ключ магазина — `new_id`, таргет — `РТО`.

## Воспроизводимость
Для призовых мест требуется код, воспроизводящий решение (см. задачу «В. Посылка итогового решения»).
Фиксируйте `random_state`, версии библиотек (`requirements.txt`) и пайплайн в `src/`.

## Старт
Окружение уже создано в `.venv/` (Python 3.13.13). Используйте его:
```powershell
.venv\Scripts\python.exe -u src\...        # запуск скриптов
.venv\Scripts\python.exe -c "import torch; print(torch.cuda.is_available())"
```
Пересоздание с нуля — см. [CLAUDE.md](CLAUDE.md) (torch ставится отдельно с CUDA-индекса cu128).

## Замечания по валидации
- Метрика на лидерборде считается лишь по части магазинов — не переобучайтесь под неё.
- Делайте **time-based** валидацию (например, обучение до 2025-01, валидация на 2025-02),
  чтобы имитировать прогноз на будущий месяц.
